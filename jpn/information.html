Future work

Jekyll & Wordpress plugin

必要であれば Jekyll と WordPress プラグインを公式のフォーラム等で公開することは考えていますが時期は未定です。 2重3重に管理するコストを考えると、早急にに公開の場を拡大する必要はないと考えています。 公開するときは、大きな更新が必要なくなった時になるでしょう。

Java & Python ver.

Java ver. と Python ver. の開発を開始する可能性があります。 その場合、恐らく Java の開発が先になります。今のところ、これ以外の言語へ対応する予定はありません。 C かあるいは C++ に対応するかもしれませんが、約束はできません。
